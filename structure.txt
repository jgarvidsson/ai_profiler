DESCRIPCIÓN DE ARCHIVOS PRINCIPALES
====================================

src/hardware_profiler.py
------------------------
- Clase HardwareProfiler
- Detecta CPU, RAM, GPU, VRAM, disco
- Genera reportes de hardware

src/model_database.py
---------------------
- Clase ModelDatabase
- Contiene base de datos de modelos de IA
- Métodos para actualizar desde repositorio

src/recommendation_engine.py
----------------------------
- Clase RecommendationEngine
- Evalúa compatibilidad hardware-modelo
- Genera recomendaciones personalizadas

src/main.py
-----------
- Punto de entrada principal
- Interfaz de usuario en consola
- Exportación de resultados

docs/
-----
- Documentación completa del proyecto
- Guías de instalación y uso
- Referencia técnica

tests/
------
- Pruebas unitarias con pytest
- Cobertura de código

data/
-----
- Almacenamiento persistente
- Cache para mejorar rendimiento

scripts/
--------
- Herramientas auxiliares
- Actualización y exportación

REQUISITOS DE INSTALACIÓN
==========================

Sistema operativo:
- Linux (Ubuntu 20.04+)
- Windows 10/11
- macOS 11+

Python:
- Python 3.8 o superior

Dependencias principales:
- psutil: Información del sistema
- GPUtil: Información de GPU
- pynvml: API de NVIDIA
- requests: Peticiones HTTP

Hardware recomendado:
- Mínimo: 4GB RAM, CPU de 4 núcleos
- Recomendado: 16GB RAM, GPU con 8GB VRAM
- Para modelos grandes: 32GB+ RAM, GPU con 16GB+ VRAM

USO BÁSICO
===========

1. Instalar dependencias:
   pip install -r requirements.txt

2. Ejecutar el perfilador:
   python src/main.py

3. Exportar resultados:
   Opción disponible en el menú interactivo
